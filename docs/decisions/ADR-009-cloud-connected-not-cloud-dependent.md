# ADR-009-cloud-connected-not-cloud-dependent

## Estado
Aceptado

## Contexto
OmniTerminal requiere decisiones estructurales para escalar sin romper el MVP ni degradar seguridad.

## Decisión
Se adopta el principio cloud-connected/no cloud-dependent para garantizar operación parcial offline y continuidad en entornos restringidos.

## Consecuencias
- Alinea producto, arquitectura y seguridad.
- Reduce riesgo operativo y técnico.
- Mejora mantenibilidad y escalabilidad incremental.
